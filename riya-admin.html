Studio Admin · manage images

Swap any image on your website, then download the updated index.html and re-upload to your hosting.

How this works:
  1. Upload a photo (or paste an image URL) for any slot below.
  2. Click Save draft to keep your changes on this device.
  3. Click Preview to see what the live site will look like.
  4. Click ⬇ Download index.html to get the updated website file.
  5. Upload that file to Hostinger / Netlify, replacing the old one.

Image tip: uploaded photos get auto-compressed for fast loading. For very large galleries, hosting images on imgbb.com (free) and pasting URLs keeps your website file smaller. This admin file stays on your computer — never upload it to your hosting.
